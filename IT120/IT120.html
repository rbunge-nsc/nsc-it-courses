This page illustrates networking technologies needed to host a web site like this one.

To reach a website, you need it's Uniform Resource Locator (URL). The URL for this site is:

https://mltv.s3.amazonaws.com/index.html

URLs are composed of different elements. For this URL, the elements are:

  • Scheme: "https"
  • Subdomain: "mltv"
  • Subdomain: "s3"
  • Second-level domain: "amazonaws"
  • Top-level domain: "com"
  • Web page: "index.html"
Reference: The Parts of a URL: A Short & Sweet Guide
